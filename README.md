# hitEnlarged
扩大控件响应区域

这是一个view的category，主要是为了扩大view的点击或交互区域，并且能不受父视图的约束，想怎么扩大就怎么扩大。

主要解决：

        1、点击区域太小，交互不灵敏问题

        2、子试图太大，超出父视图范围导致无法响应点击事件问题。
